# Template: Notifications / Alerts

> Spec ref: Component Specification §11

## Notification Dropdown (bell icon in top nav)

```
+-------------------------------+
| 🔔 Notifications (3 unread)   |
+-------------------------------+
| ● Notification 1  [timestamp] |
| ● Notification 2  [timestamp] |
|   Notification 3  [timestamp] |
+-------------------------------+
| [Mark all as read] [View all] |
+-------------------------------+
```

## Toast / Snackbar States

```
[✓ Success: Action completed]       ← success toast
[⚠ Warning: Something to note]      ← warning toast
[✗ Error: Something went wrong]     ← error toast
[ℹ Info: Here's some information]   ← info toast
```
